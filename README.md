# Terraform Learning & Practice Repository

This repository contains **Terraform Infrastructure as Code (IaC) examples** created while learning and practicing Terraform with AWS. It is designed for **beginners to intermediate DevOps learners** who want hands-on experience with real-world Terraform concepts such as modules, loops, provisioners, remote backends, and multi-environment setups.
